# Rutificador Landing Page Plan

## 1. Concept and Content
*   **Target Audience:** Chile
*   **Topic:** "Rutificador" - a tool for looking up information (like name, address) using the Chilean unique tax/identification number (RUT - Rol Único Tributario).
*   **Purpose:** Create a beautiful, SEO-optimized landing page for cloud stacking backlink purposes. The content should be general and positive about the utility of a "rutificador" service, focusing on its convenience and importance for identity verification in Chile.
*   **Key Content Sections:**
    *   **Hero:** Catchy headline, brief description of what a "rutificador" is, and a call-to-action (even if it's just a placeholder link).
    *   **Features/Benefits:** List the benefits of using a rutificador (e.g., identity verification, finding contact info, business checks).
    *   **SEO Section (Backlink Anchor):** A section with a brief, high-quality paragraph that naturally includes the desired anchor text for the dofollow backlink.
    *   **Footer:** Copyright, disclaimer (e.g., data is public, not responsible for misuse).

## 2. Design and Aesthetics
*   **Color Scheme (Chilean Flag):**
    *   **Blue (Philippine Blue):** `#0039A6` (or `#043696` or `#0039a6`) - Represents the sky and the Pacific Ocean.
    *   **Red (Maximum Red):** `#D52B1E` (or `#D52B1E`) - Represents the blood spilled for independence.
    *   **White:** `#FFFFFF` - Represents the snow of the Andes.
*   **Logo:** A simple, modern logo for "Rutificador" will be generated in the next phase. It should incorporate the Chilean colors.
*   **Overall Vibe:** Professional, trustworthy, and modern.

## 3. SEO Requirements
*   **Target Keyword:** "rutificador", "rutificador chile", "buscar rut"
*   **Title Tag:** Must be compelling and include the primary keyword. E.g., "Rutificador Chile Oficial: Consulta RUT y Nombre al Instante"
*   **Meta Description:** E.g., "La herramienta más rápida y confiable para buscar personas y empresas en Chile por RUT o nombre. Verificación de identidad simplificada."
*   **H1 Tag:** Must contain the primary keyword.
*   **Content:** High-quality, relevant content that naturally uses the keywords.
*   **Backlinks:** The user intends to add 1-2 dofollow backlinks from this page. The design should include a natural place for these, possibly in a "Resources" or "Partners" section.

## 4. Technical Details
*   **Deployment:** GitHub Pages (static HTML, CSS, JS).
*   **Structure:** Simple, single-page HTML. Use modern CSS (Flexbox/Grid) for a beautiful, responsive layout.
*   **Backlink Implementation:** The user will manually add the dofollow links. The code will include a placeholder for this.
*   **Logo Implementation:** Will use a generated image and save it as `logo.png`.
*   **Favicon:** Will use a simple favicon, possibly a star or a stylized 'R' in Chilean colors.
